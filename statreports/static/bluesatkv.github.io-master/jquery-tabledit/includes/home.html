Key Features

Inline Editor
It is possible to edit inline like a spreadsheet or with buttons to change between edit and view mode.
Columns
You must choose a table column to use as identifier column and choose the ones you want to turn into editable columns.
Edit, Delete and Restore
Edit the pre-selected columns with button to save or delete a row with confirmation button and the possibility of restoring it.
Compatible with Bootstrap
It is used some Bootstrap components, glyphicons and CSS classes.
Customizable
Possibility of changing the CSS classes and contents of buttons and other things.
Examples and Documentation
On this site you can see some examples with the corresponding source code and its documentation.

Getting started with Select2

1. Download Jquery-Tabledit

2. Include the following lines of code in the your HTML.

<script src="your_folder_with_jquery-tabledit/jquery.tabledit.js"></script>

3. Initialize Jquery Tabledit on the <table id="my_table"> element that you want

$('#my_table').Tabledit({
  url: 'ajax/example.php',
  columns: {
      identifier: [0, 'id'],
      editable: [[1, 'nickname'], [2, 'firstname'], [3, 'lastname']]
  }
});

4. Check out the examples page to start using the additional features of Jquery-Tabledit.
